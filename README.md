<a name="stack"></a>
## 1. Tech stack
Language: Kotlin  
DI: Koin  
Architecture: MVVM  
Multithreading: Coroutines  
Layout: Legacy View System with Viewbinding and ConstraintLayout  
Images loading: Glide  
HTTP client: Retrofit  
Database: Room
